#_Como Preparar Chifrijo_
##ingredientes:

1. Arroz
2. Cubaces
3. Chicharon
4. Chimichurri
5. Aguacate
6. Piquitos
7. Chile
8. Birra

##Procedimiento:

- Se pica en trozos el chicharron
- Se mezcla con el arroz y los cubaces
- se agrega el aguacate y el chimichurri
- **se agrega el chile**
- se colocan los **_piquitos_**
- _se destapa la **birra**_
